# Getting started

Pages are MXML files containing a `<s:Page>` tag that explicitly sets the path rule.

Here is an example:

```mxml
<!-- src/com/example/pages/Home.mxml -->
<?xml version="1.0"?>
<s:Page xmlns:s="http://skyflex.com/1.0" path="/">
    <s:Label text="Hello world!"/>
</s:Page>
```
